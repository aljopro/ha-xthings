# Device Handler Types Reference

> Source: https://developer.xthings.com/hc/en-us/articles/39872638936217-Device-Handler-Types-Reference

---
Every device using Xthings Schema must be associated with a device handler type. This indicates the capabilities available on the device.

📌 IMPORTANT

The Health Check is mandatory for each Device Profile.
